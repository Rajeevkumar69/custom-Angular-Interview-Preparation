Change Detection in Angular

Change Detection is how Angular knows when UI updated.

⚙️ How Angular Does It
                    {{ userList.length }}

When userList changes in the component, Angular detects the change and updates the DOM automatically. 

                    this.userList = 20;
Interview One-Liner: Change Detection is Angular’s mechanism to keep the UI in sync with data. It use to update the UI as soon data got updated, or interact with UI.
