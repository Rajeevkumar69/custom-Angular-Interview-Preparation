View Encapsulation In Angular

View encapsulation in Angular is a mechanism that controls how the CSS styles defined within a component are applied to the DOM, ensuring that a component's styles are scoped and isolated from the rest of the application. This prevents style conflicts and promotes modular, reusable components.

  • Emulated (Default)

    Styles are scoped to the component and do not leak outside, but the DOM is not truly isolated.

  • ShadowDom

    Uses the browserâ€™s native Shadow DOM, ensuring complete style isolation within the component.

  • None

    No encapsulation is applied. Styles behave as global styles and may affect other components.
